form(모양, 형식) 태그

- 입력 양식을 작성하는 판(영역)

- 내부에 작성된 값을 
    지정된 경로(html, 서버)로
    제출하는 역할


- 사용예시
    : 회원가입, 로그인, 글 작성, 검색 ... 


[속성]
1) action(행동) : 제출될 파일 또는 서버 경로 작성

2) method(방식) : 제출방식을 지정

3) name : 이름지정 (JS)

4) target : 제출된 페이지를 현재 탭 or 새탭 어디서 볼지 결정

button 관련 태그

- 버튼을 만드는 태그

- type 속성에 따라서 용도가 달라짐

type="submit"
- form태그 내부에 작성된 경우
    해당 form에 모든 input 값을 action 경로로 제출하는 버튼 (form 제출)

type="reset"
- form태그 내부에 작성된 경우
    해당 form의 모든 input 값을 제거하는 버튼(초기화)

type="button"
