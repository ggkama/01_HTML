김태윤의 페이지

ul (Unordered List) 태그 : 순서가 없는 목록

li (List item) 태그 : 목록의 한줄

  • 맘스터치
  • 라면
  • 김밥
    • 야채
    • 참치
    • 고추
      • 청양
      • 오이
      • 꽈리
  • 샌드위치
  • 국밥

ol(Ordered List) : 순서가 있는 목록

  1. 우육면
  2. 김밥
  3. 쉑쉑버거
*type 속성 : 표시되는 숫자 형식 지정

type="a" : 엉어 소문자(a,b,c,d,e....)
type="A" : 영어 대문자(A,B,C,D,E....)
type="i" : 로마 소문자(i, ii, iii, iv, v, vi...)
type="I" : 로마 대문자(I, II, III, IV, V, VI...)
type="1" : 숫자 (1, 2, 3) 
                (기본값 = 아무것도 적지 않았을 때 적용)
    
        * start 속성 : 목록 시작 번호
        * reversed 속성 : 순서 반대로(역순)
  1. HTML5
  2. CSS3
  3. JavaScript
  4. Java
  5. DB(Oracle)

Emmet : HTML, CSS 빠른 자동완성 자동 플러그인

태그명 : 태그 시작/종료 태그 자동완성

> : 태그 내부에 작성(하위 생성)

+ : 태그 다음에 작성(형제)

*숫자 : 숫자 만큼 생성

{} : 태그 내부에 작성될 내용

$ : 1부터 1씩 증가하는 숫자(시퀀스)
  1. 아이템 <1>
  2. 아이템 <2>
  3. 아이템 <3>
  4. 아이템 <4>
  5. 아이템 <5>
